“Find Courses” — drop-in feature for fcei.eu

Click Find Courses. The search header and footer stay fixed; the 18 areas scroll inside the panel (custom emerald/moss scrollbar, keyboard scrollable). Colours and the Bricolage / Hanken / Space Mono type follow the FCEI system.

Built to add to the fcei.eu menu, not rebuild it. Copy the <style> block (incl. the :root tokens), the <div class="fc">…</div> block and the <script> block into the site's header, beside the existing search.

How it maps to your real pages (fcei.eu is a hash-routed SPA): on load the menu adopts the real href from your existing nav by matching each item's label, so it links to the same routes the site already uses — no manual mapping. For any item it can't auto-match, set window.FCEI_ROUTES[slug] (template above) or edit the FC_ITEMS hrefs. Open the console: it logs how many of the 18 are mapped, and findCoursesAudit() prints the full label → destination table. Confirmed real links on the site today: https://fcei.eu/login.html and https://fcei.eu/booking.html.
