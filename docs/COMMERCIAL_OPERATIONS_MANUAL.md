# Commercial Operations Manual

## Daily operations
- Check new bookings
- Check orders and failed payments
- Respond to school demo requests
- Review new user registrations
- Review intervention-case demo submissions

## Weekly operations
- Add or improve item-bank questions
- Upload new resources
- Review assessment results
- Update course content
- Check backups

## Monthly operations
- Review revenue by product
- Review course completion data
- Review support requests
- Publish one new resource or webinar
- Check legal/compliance changes

## Support inbox categories
- Payment issue
- Login issue
- Course access issue
- Certificate request
- Institutional licence enquiry
- Refund request
- Safeguarding/data concern
