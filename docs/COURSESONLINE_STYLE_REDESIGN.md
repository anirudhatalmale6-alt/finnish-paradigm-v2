# CoursesOnline-Inspired Redesign Notes

This redesign changes The Finnish Paradigm Stripe Live Ready website from a simple course site into a course marketplace-style CPD platform. It is structurally inspired by public course comparison patterns: prominent search, course navigator, categories, study method filters, qualification levels, popular courses, resource/newsletter sections, and cookie notice. It does not copy CoursesOnline branding, text, or proprietary assets.

## Key UI Changes
- New top strip and sticky marketplace navigation.
- Search-first header with subject, study method, qualification level, job-role and provider/partner links.
- Course navigator engine for role/goal/time recommendations.
- Course catalogue with sidebar filters, cards, compare drawer and Stripe checkout hook.
- Popular subjects, course levels, resource/newsletter and certification sections.
- Refined CPD metadata and learner/admin dashboard previews.

## Colour Scheme
- Navy foundation for trust and education authority.
- Bright blue/cyan for search and action flows.
- Pink/orange accents for modern marketplace energy.
- White cards and soft blue backgrounds for catalogue readability.

## Navigation Tools
- Sticky header.
- Search bar.
- Mega-category strip.
- Breadcrumbs.
- Course filters.
- Course comparison drawer.
- Course navigator recommendation tool.
- Dashboard/admin quick routes.

## Engine Updates
- JavaScript catalogue filtering.
- Recommendation engine based on role, goal and time.
- Course comparison state.
- Stripe checkout API hook.
- Auth-aware navigation.
- Admin, booking, assessment and resource API integrations retained.
