# Content Architecture Map

This folder contains the canonical content used by the web app and PDF resource packs.

## Core suites
- Product suites: Teacher Starter, Early Intervention, School Leadership, Retired Mentor, Enterprise License
- Course catalogue: FCFP, ASTI, SELD
- Toolkits: classroom, intervention, management, retired teachers, parent communication, AI prompt vault
- Assessments: adaptive item bank seeded in SQLite on first startup
- Videos: scripts and AI presenter prompts for external AI-avatar tools

## Avoiding repetition
The same ideas appear once in the source data and are reused by the front end, API, PDF documents, and admin dashboard.
